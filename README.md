# Gelişmiş Discord Destek Botu

Bu proje, `discord.js v14` ile yazılmış gelişmiş bir destek botudur.

## Özellikler

- Seçim menülü destek paneli
- Kategori bazlı özel ticket açma
- Embed log sistemi
- Ticket kapanışında otomatik yorum/puan DM’i
- Yetkili tarafından ticket üstlenme
- Ticket içinden tek butonla üye yönetimi
- Ticket kapanışında transkript oluşturma
- Kapanış sonrası kullanıcıdan yorum/puan alma
- Yetkililer için manuel embed DM gönderme komutu
